Bible
=====

哥自己使用的 Android 版圣经。

其实市场上已有的 Android Bible应用已经很多了，但用起来都不爽，怎么办？向他们提需求吗？瞎扯蛋！最后脑袋一抽筋：自己开发一个 bible app，这样哥想怎么搞就怎么搞？

圣经的数据很大的哦！还好，哥一年前就把这些数据搞好了，mysql的。
